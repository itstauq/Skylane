# Goal

Focus widget for committing to one finish line for the current block.

## What it does

- asks for one main goal
- lets the user choose a lightweight time box or keep the block open
- flips into a single active commitment card once the goal is set
- shows elapsed and remaining time when a time box is active

## Notes

- The visual design is optimized for the active state, not the setup form.
- The active goal persists in local storage.
- It is bundled with Skylane, but not yet seeded into a default layout.
